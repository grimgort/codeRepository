SFML Utils
==========

It provide some adds to the SFML library including:

* Resource manager
* Event manager with callback
* A Gui
 * Frame
 * Button
 * Label
 * Vertical layout
* Box2D debugDraw (optional)
* 2D map framwork
* Entity system

Requirements
------------

* SFML 2.0 +
* Box2D 2.3 + (optional)
* cpp-utils : json parser (for map framwork, https://github.com/Krozark/cpp-utils)
* C++11 compiler
