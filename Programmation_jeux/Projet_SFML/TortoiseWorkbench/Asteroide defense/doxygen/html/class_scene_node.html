Description détaillée

Classe representant le sceneNode.

La classe gère les noeuds représantant l'ensemble des objets du jeux. Par exemple, les avions, les projectiles, les pickup sont hérite des sceneNode Les méthodes de cette classe permettent notamment de gérer la création et la destruction des objets (ou noeud).

L'architecture des noeuds est un arbre. C'est à dire qu'il y a un noeud principal qui posséde plusieurs branche avec d'autres noeud, etc...
